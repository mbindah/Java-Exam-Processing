# Java Exam Processing System
This is a Java program that processes university examination results.

## Features
- Reads student details (Name, Student ID)
- Accepts scores for 5 subjects:  
  - Object Oriented Programming 1  
  - Calculus 1  
  - Database Systems  
  - Artificial Intelligence  
  - Ethics  
- Calculates the average grade based on Strathmore University's grading system
- Displays a detailed report card with recommendations

## How to Run the Program
1. Open a terminal or command prompt.
2. Navigate to the folder containing the Java files.
3. Compile the program using:
   ```sh
   javac Main.java
   ```
4. Run the program using:
   ```sh
   java Main
   ```
